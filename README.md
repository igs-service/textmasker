# TextMasker

**簡単秘匿情報マスキング**

業務で扱う機密情報を、ブラウザ内で安全にマスキングします。
外部へのデータ送信は行っておりませんので、ローカルにダウンロードすればインターネット接続がなくても動作します。

## デモサイト

<a href="https://igs-service.github.io/textmasker/" target="_blank" rel="noopener">https://igs-service.github.io/textmasker/</a>

## 主な機能

### マスキング方式

- **仮名化**: `{{IPv4:001}}` のように連番付きラベルで置換（必要に応じてランダム文字列を付与）
- **マスク解除**: マスク済みテキストを元の情報に戻す機能

### 対応パターン（20種類以上）

- IPv4/IPv6アドレス
- MACアドレス
- メールアドレス
- 電話番号
- クレジットカード番号
- 日本の郵便番号
- URL/ドメイン
- 日本の住所（都道府県・市区町村）
- その他多数

### カスタマイズ機能

- **任意辞書**: 独自のマスキングルールを追加
- **テキスト辞書**: TXT/CSVファイルから辞書を一括登録
- **住所辞書**: [Geolonia 住所データ](https://geolonia.github.io/japanese-addresses/)（CC BY 4.0）を使用
- **設定保存**: マイ設定をJSON形式で保存・読み込み

## 使い方

1. デモサイトにアクセス
2. 「入力」欄にテキストを貼り付け
3. 「マスク実行」ボタンをクリック
4. マスキングされたテキストが「出力」欄に表示されます

「基本設定」「任意辞書」「テキスト辞書」「住所辞書」タブで細かくカスタマイズできます。

## ローカルで動かす（ZIP/clone）

ブラウザの制限により `file://` で直接開くと CSS/JS が読み込めません。
※解凍したフォルダで python を実行できる必要があります。

### [Windows]

```bat
python -m http.server 8000
```

`http://localhost:8000/` にアクセスします。

### [Mac/Linux]

```bash
python3 -m http.server 8000
```

`http://localhost:8000/` にアクセスします。

## プライバシー

- **完全クライアントサイド処理**: すべての処理はブラウザ内で完結
- **外部送信なし**: 入力したテキストは外部サーバーに送信されません
- **オフライン動作**: ダウンロードすればインターネット接続不要（住所辞書更新を除く）
