# TextMasker

**簡単秘匿情報マスキング**

業務で扱う機密情報を、ブラウザ内で安全にマスキングします。
外部へのデータ送信は行っておりませんので、ローカルにダウンロードしたりインターネット接続がなくても動作します。

## デモサイト

<a href="https://igs-service.github.io/textmasker/" target="_blank" rel="noopener">https://igs-service.github.io/textmasker/</a>

## 主な機能

### マスキング方式

- **仮名化**: `<IPv4_001>` のように連番付きラベルで置換
- **伏せ字**: `****` や任意の文字列で置換

### 対応パターン（20種類以上）

- IPv4/IPv6アドレス
- MACアドレス
- メールアドレス
- 電話番号
- クレジットカード番号
- 日本の郵便番号
- URL/ドメイン
- 日本の住所（都道府県・市区町村）
- その他多数

### カスタマイズ機能

- **任意辞書**: 独自のマスキングルールを追加
- **住所辞書**: [Geolonia 住所データ](https://geolonia.github.io/japanese-addresses/)（CC BY 4.0）を使用
- **設定保存**: マイ設定をJSON形式で保存・読み込み

## 使い方

1. デモサイトにアクセス
2. 「入力」欄にテキストを貼り付け
3. 「マスク実行」ボタンをクリック
4. マスキングされたテキストが「出力」欄に表示されます

「基本設定」「任意辞書」「住所辞書」タブで細かくカスタマイズできます。

## プライバシー

- **完全クライアントサイド処理**: すべての処理はブラウザ内で完結
- **外部送信なし**: 入力したテキストは外部サーバーに送信されません
- **オフライン動作**: ダウンロードすればインターネット接続不要（住所辞書更新を除く）
